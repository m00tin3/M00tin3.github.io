学前必知概念

  • 三层架构：Controller（接收请求）→ Service（业务逻辑）→ Mapper（数据库操作）
  • IoC 控制反转：对象由 Spring 容器创建和管理
  • DI 依赖注入：通过 @Autowired 自动注入依赖
  • AOP 面向切面编程：在不修改原有代码的情况下添加公共功能
  • RESTful API 设计：使用 HTTP 方法（GET/POST/PUT/DELETE）表示操作
  • JSON 序列化/反序列化：前后端数据交互格式
  • Maven 依赖管理：自动化构建和依赖管理
